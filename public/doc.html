Bitcoin Data API

Explore the Bitcoin Data API and retrieve information about various cryptocurrencies.

Endpoints

HTTP Method Endpoint Description
GET /api/cryptocurrency Rettrives data for all cryptocurrency we provide
GET /api/cryptocurrency/{symbol} Retrieves data for a specific cryptocurrency based on the symbol you can find in the Coin List.

Example

Retrieve data for Bitcoin using the following example code:


1   //Define the variables of your html div where you want to display the DATA
2   let bitcoinDiv = document.getElementById('bitcoin');
3   let bitcoinLogo = document.getElementById('bitcoin-logo');
4
5   //Define the URL and setup the response structure
6    fetch('http://209.38.254.29/api/cryptocurrency/btc')
7       .then(response => response.json())
8       .then(data => {
9
10       //Display the JSON DATA here
11         let bitcoinJsonName = data.name;
12         let bitcoinJsonLogo = data.logo;
13
14         let image = document.createElement('img');
15         image.src = bitcoinJsonLogo;
16         bitcoinDiv.innerHTML = bitcoinJsonName;        
17         bitcoinLogo.appendChild(image);
19          });
20
21    //Catch for errors
22    .catch(error => { console.error('Error fetching data:', error);
23      });
24

Response

The API returns JSON data containing information about the requested cryptocurrency, including name, symbol, logo, and description.

Error Handling

If the requested cryptocurrency is not found, the API returns a 404 error.

Rate Limiting

This API currently does not implement rate limiting.
